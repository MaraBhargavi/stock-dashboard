# 📈 Real-Time Stock Market Dashboard

A real-time interactive stock market dashboard built using Python and Streamlit.  
It provides live stock prices, technical indicators, multi-stock comparison, and latest financial news.

---

## 🌐 Live Demo
👉 https://stock-dashboard-snwaxl9k5oswmuxue68k9s.streamlit.app

---

## 📌 Features

- 📊 Real-time stock price tracking
- 📈 Multi-stock comparison (AAPL, TSLA, MSFT, etc.)
- 📉 Technical indicators (SMA - Simple Moving Average)
- 🔄 Auto-refresh every 5 seconds
- 📰 Live financial news feed
- 📊 Interactive charts and graphs

---

## 🛠️ Technologies Used

- Python 🐍
- Streamlit (Web Framework)
- yfinance (Stock Data API)
- pandas (Data Analysis)
- Plotly (Data Visualization)
- feedparser (News Feed)
- streamlit-autorefresh (Auto Refresh)
